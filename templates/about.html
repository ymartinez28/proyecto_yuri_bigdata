Descripción:

DocuSearch es una aplicación web diseñada para facilitar la búsqueda inteligente y eficiente de documentos digitales indexados en Elasticsearch. Ofrece una experiencia ágil y precisa. Está pensada especialmente para proyectos académicos y de analítica de datos, integrando una interfaz clara, navegación amigable y tecnologías modernas como Flask y MongoDB.

DocuSearch permite subir, indexar y buscar documentos mediante Elasticsearch. Usa una arquitectura basada en Flask, con MongoDB para almacenamiento adicional y filtros avanzados para mejorar la experiencia del usuario.

Esta aplicación fue desarrollada como parte de un proyecto de Maestría en Analítica de Datos por Yuri Martínez en la Universidad Central.
